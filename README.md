# Fitness Influencer Coaching Platform - ER Diagram

This project contains the ER diagram design for an online fitness coaching platform.

The system models how trainers manage clients, offer different coaching plans, handle subscriptions, schedule sessions, track client progress, and manage payments.

## Key Features

* Trainers can manage multiple clients
* Clients can subscribe to different plans over time
* Plans include consultation, live sessions, and coaching
* Sessions are scheduled between trainers and clients
* Progress tracking includes weight, measurements, and notes
* Payments are linked to subscriptions

## Tech / Tool Used

* DBML (dbdiagram.io)
